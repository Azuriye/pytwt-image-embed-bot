# pytwt-image-embed-bot
Watches channel for Twitter links and embeds images with links to Tweet URL

## How To

### Install

You need the following: 

 1. Python 3.9.2 
 2. discord.py 
 3. gallery-dl
 4. aiohttp
 5. ffmpeg
